# 打开 about:config



在[地址栏](https://wikipedia.org/wiki/Address_bar#Firefox)，输入**about:config**，然后按回车。

- 可能出现**这可能使质量保证失效！**警告页面。点击**我了解此风险！**，以继续打开about:config页面。
